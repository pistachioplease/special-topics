What are web development frameworks?

Software providing generic functionality that can be selectively changed by additional user-written code, thus providing application-specific software.

"Don't reinvent the wheel."

Not yet.

Web Dev Framework is a subset of Software Framework

What is a Software Framework?

  • provides a standard way to build and deploy applications
  • facilitate development of software applications, products and solutions

Key Distinguishing Features of a Framework

  • inversion of control, "overall program's flow of control is not dictated by the caller, but by the framework"
  • extensibility, programmers can add specialized user code to provide specific functionality
  • users can extend the framework, but should not modify its code
