What is a recommendation engine?

Approaches to Recommendation Engines

Collaborative Filtering

Collecting and analyzing large information on user’s behaviors, activities or preferences and predicting what users will like based on similarity to other users.

"People who agreed in the past, will agree in the future."

Content-Based Filtering

Based on a description of the item and a profile of the user’s preference.

Keywords are used to describe the items and a user profile is built to indicate the type of item this user likes.

Hybrid Filtering

Collaborative: Collaborative systems locate peer users / items with a rating history similar to the current user or item and generate recommendations using this neighborhood.

Content-based: The system generates recommendations from two sources: the features associated with products and the ratings that a user has given them.

The website makes recommendations by comparing the watching and searching habits of similar users (i.e., collaborative filtering) as well as by offering movies that share characteristics with films that a user has rated highly (content-based filtering).
